ДЕФЕКТ №1: Метод capitalize() неправильно обрабатывает строки с пробелами в начале
-------------------------------------------------------------------------------
ID: DEF-001
Приоритет: Средний
Серьезность: Незначительная
Компонент: StringUtils.capitalize()
Автор: Команда тестирования
Дата: 2026-06-26

Описание:
  Метод capitalize() работает некорректно со строками, начинающимися с пробела.
  Согласно документации и ожидаемому поведению, метод должен делать заглавной
  первую букву строки (после пробела), но фактически этого не происходит.

Шаги воспроизведения:
  1. Создать экземпляр класса StringUtils
  2. Вызвать метод capitalize() с аргументом " текст"
  3. Получить результат " текст"

Ожидаемый результат:
  " текст" - пробел сохраняется, первая буква становится заглавной

Фактический результат:
  " текст" - пробел сохраняется, первая буква остается в нижнем регистре

Воспроизводимость: Всегда
Окружение: Python 3.14, Windows 11

Примечание:
  Использование встроенного метода str.capitalize() приводит к такому поведению.
  Рекомендуется использовать метод str.title() или реализовать собственную логику.

ДЕФЕКТ №2: Метод trim() удаляет только обычные пробелы
-------------------------------------------------------------------------------
ID: DEF-002
Приоритет: Средний
Серьезность: Значительная
Компонент: StringUtils.trim()
Автор: Команда тестирования
Дата: 2026-06-26

Описание:
  Метод trim() удаляет только обычные пробелы (символ ' '), но игнорирует другие
  пробельные символы: табуляцию, перевод строки, unicode пробелы и т.д.
  Это может привести к неожиданному поведению при работе с различными входными
  данными, особенно при парсинге текста из внешних источников.

Шаги воспроизведения:
  1. Создать экземпляр класса StringUtils
  2. Вызвать метод trim() с аргументом "\tтекст" (с табуляцией)
  3. Получить результат "\tтекст"

Ожидаемый результат:
  "текст" - все пробельные символы должны быть удалены

Фактический результат:
  "\tтекст" - табуляция сохраняется

Воспроизводимость: Всегда
Окружение: Python 3.14, Windows 11

Примечание:
  Рекомендуется использовать встроенный метод str.lstrip() без аргументов,
  который удаляет все пробельные символы по умолчанию.
